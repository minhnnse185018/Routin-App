# Assets Directory

Place your app assets here:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

## Required Sizes

### icon.png
- Size: 1024x1024px
- Format: PNG with transparency
- Use the Routin logo with the lime green accent

### splash.png
- Size: 2048x2732px (or similar high resolution)
- Format: PNG
- Background: #161616 (dark gray)
- Center: Routin logo

### adaptive-icon.png
- Size: 1024x1024px
- Format: PNG with transparency
- Safe zone: 512x512px center
- Background: #161616

### favicon.png
- Size: 48x48px
- Format: PNG
- Simple Routin logo icon
